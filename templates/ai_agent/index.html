Quick Setup Guide
Prerequisites
  • Place agent.py in blueprints/
  • Configure environment variables in .env
  • Ensure Azure OpenAI and Speech API keys are set
  • Configure Asterisk ARI credentials
Dialplan Configuration

Add to your Asterisk dialplan:

exten => 7000,1,NoOp(AI Agent) same => n,Stasis(ai-agent) same => n,Hangup()
Tip: The AI agent runs as a standalone process for better stability and isolation.
